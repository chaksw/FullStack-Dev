Selection with jQuery

You can use the $ to select elements from the DOM, it will fell a lot like selection using querySelectorAll()

For example:

  • Selecting all 'a' Tags:
    • $('a')
  • Selecting all element from the class "container":
    • $('.container')
  • Selecting all elements with id "special":
    • $('#special')
